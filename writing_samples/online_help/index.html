Online Help

Overview

Here are “What’s New,” conceptual, and procedural topics that I wrote for Microsoft Dynamics AX 2012. Microsoft Dynamics AX, recently rebranded as Dynamics 365, is a multi-government, multi-language, multi-currency global enterprise resource planning (ERP) software suite. Dynamics AX is part of the Microsoft Dynamics family.

When I worked at Tyler Technologies, we partnered with Microsoft to develop public sector accounting features for Microsoft Dynamics AX 2012 that addressed the unique needs of public sector organizations. Tyler Technologies is a world-class leader in creating public sector software solutions and services. You can read more about the Tyler and Microsoft partnership here: Microsoft Dynamics AX for the Public Sector.

As the User Assistance Lead for the Tyler writing team, I wrote online help for the Accounts Payable, Vendors, and General Ledger modules. I also wrote all of the related user interface text, such as field labels, buttons, tooltips, and error messages.

The complete online product help is available at the Microsoft Dynamics AX 2012 Technical Library.
